WebSocket client for 1999.io

This app hooks up with the 1999.io server using WebSockets. Every time a message is created or updated, the socket returns with the content of that message.

The sockets time out after approximately 60 seconds. When that happens, we fire up another WebSocket call. And, if a call doesn't return after 75 seconds, we start another call, assuming we missed the timeout message from the server.
